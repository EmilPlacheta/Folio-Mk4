Propertier

This React app is designed to help manage multiple properties, by giving the user a condensed overview of the current properties they manage. Easily add, remove or change status of the property (rented, renovating, sold etc) and see the total income from all rent, across all properties. It enables the user to have easy access to checklists used when inspecting the property, or to download documents e.g tenancy agreement. Materialize framework was used to provide a familiar ‘Google’ look and mobile responsiveness, as the app will mostly be used on tablets. Propertier is a full stack MERN app (built with MongoDB, Express, React and Node.js) with a complete back-end and its own API.
